# Code of Conduct

## Standards

This project's discussion spaces — issues, pull requests, and any affiliated forums — are for technical work. Expected behavior:

- Communicate clearly and directly about technical matters
- Accept and give critique on code and design, not on people
- Acknowledge when you're wrong; update your position when presented with evidence
- Stay on topic

Behavior that will not be tolerated:

- Harassment, personal attacks, or discriminatory language
- Deliberate intimidation or sustained disruption of discussion
- Publishing others' private information without consent
- Any conduct that would be considered inappropriate in a professional engineering context

---

## Enforcement

Maintainers have discretion to remove comments, close issues, or revoke contribution access in response to violations. Repeated or severe violations will result in a permanent ban.

Reports of misconduct can be sent to: **conduct@gearsec.io**

Reports are handled confidentially. Retaliation against reporters is itself a violation.

---

## Scope

This Code of Conduct applies within all project spaces. It does not govern participants' conduct outside those spaces unless that conduct directly affects project contributors or the project's integrity.
